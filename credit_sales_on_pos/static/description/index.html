User can do credit sales in POS.

How to use

  1. Create "Credit sales" (Identical to normal Sales Journal) payment method in POS configuration. Remember to add this payment method in the main POS session.
  2. In POS interface, click "Invoice" instead of "Validate", and a draft invoice will be printed.
  3. User can merge the draft invoices at end of cycle by using addon Account Invoice Merge Wizard
